ng-switch

Директива ng-include позволяет подгружать нужные файлы,
однако иногда рациональнее будет определить фрагменты кода html сразу же на главной странице и отображать их в зависимости от условий.
Для этого предназначена директива ng-switch.

Чтобы применить данную директиву, изменим главную страницу следующим образом:

  • {{tablet.name}}

    Цена: {{tablet.price}} $

    Производитель: {{tablet.company}}

  • {{phone.name}}

    Цена: {{phone.price}} $

    Производитель: {{phone.company}}

Выберите товары для просмотра

Использование директивы ng-switch напоминает конструкцию switch..case ,
которая есть во многих языках программирования. Директива используется вместе с атрибутом on,
который указывает на выражение. В данном случае выражением является значение объекта data.mode.

Для оценки значения data.mode используется директива ng-switch-when:
<div ng-switch-when="Tablets">
    <!--код, если data.mode Tablets -->
</div>
<div ng-switch-when="Phones">
    <!--код, если data.mode Phones -->
</div>
